# 의료 인공지능과 영상 분할을 공부하는 이승수입니다.

순천향대학교 컴퓨터공학과 학부생으로, 의료 인공지능과 의료영상 분석, 특히 Segmentation 분야에 관심을 가지고 공부하고 있습니다.

MI2RL 겨울 인턴에서는 Mask R-CNN 기반 흉부 X-ray 폐 분할 프로젝트를 수행했고, 학부 졸업논문에서는 U-Net 계열 모델을 활용한 신장 병리 이미지 사구체 분할 연구를 진행했습니다.

단순히 모델을 실행하는 것에서 끝내기보다, 데이터 전처리, 모델 학습, 정량 평가, 결과 시각화, 실패 사례 분석까지 연구의 전체 흐름을 이해하는 개발자이자 연구자가 되는 것을 목표로 하고 있습니다.
